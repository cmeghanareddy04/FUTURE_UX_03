
# CRM Dashboard UI

## Project Overview

CRM Dashboard UI is a modern and intuitive dashboard designed in Figma for small agencies and B2B service-based businesses. The interface helps users efficiently manage leads, clients, tasks, and revenue while providing clear insights through data visualization and an organized workflow.

## Key Features

• Dashboard overview with business KPIs
• Lead management
• Client management
• Revenue analytics dashboard
• Interactive charts and graphs
• Task tracking
• Recent deals table
• Search, filter, and sorting options
• Clean and modern UI
• Responsive desktop dashboard layout
• User-centered navigation
